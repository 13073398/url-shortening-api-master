## Overview

This project is a solution to the URL shortening API Challenge on Frontend Mentor. It allows users to shorten any valid URL and view a list of their shortened links. The project is built using HTML, CSS, and JavaScript.

## Features

- Shorten any valid URL
- View a list of shortened links
- Copy the shortened link to the clipboard in a single click
- Responsive design for different screen sizes

## Usage

1. Open the `index.html` file in your web browser.
2. Enter a URL in the input field and click the "Shorten It!" button.
3. The shortened link will be displayed in the "Shorten URL" section.
4. Click the "Copy" button to copy the shortened link to your clipboard.
5. Refresh the page to see the list of shortened links.
